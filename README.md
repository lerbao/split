# ADB 自动化脚本集

通过 ADB 控制 Android 设备，支持自动滑动和抖音互动。

## 前置条件

1. 安装 [ADB](https://developer.android.com/tools/adb) 并加入 PATH
2. Android 设备开启 USB 调试，通过 `adb devices` 确认已授权

---

## 1. swipe_loop.ps1 — 通用自动滑动

纯滑动脚本，适用于任何需要上下滑动的 App。

```powershell
.\swipe_loop.ps1
```

### 参数说明

| 参数 | 随机范围 | 说明 |
|------|----------|------|
| 起始 X | 480 ~ 599 | 起始横坐标 |
| 起始 Y | 1900 ~ 2299 | 起始纵坐标（屏幕底部） |
| 终点 X | 480 ~ 599 | 终点横坐标 |
| 终点 Y | 100 ~ 499 | 终点纵坐标（屏幕顶部） |
| 持续时间 | 300 ~ 799ms | 滑动动作耗时 |
| 间隔 | 10/11/12/13/14/15s | 每次滑动后等待时间 |

## 2. douyin_bot.ps1 — 抖音自动刷视频 Bot

在抖音界面自动刷视频，并随机执行互动操作。

```powershell
.\douyin_bot.ps1
```

### 动作概率

| 动作 | 概率 | 说明 |
|------|------|------|
| 点赞 | 40% | 双击屏幕中央红心 |
| 收藏 | 15% | 点击右侧收藏图标 |
| 评论 | 10% | 打开评论→输入随机文字→发送 |
| 跳过 | 35% | 仅滑动，不互动 |

### 抖音按钮坐标（1080x2400 基准，带随机偏移）

| 按钮 | 基准坐标 | 偏移范围 |
|------|----------|----------|
| 心形(点赞) | (930, 1300) | ±30px |
| 评论图标 | (930, 1550) | ±30px |
| 收藏图标 | (930, 1780) | ±30px |
| 双击点赞 | (540, 1200) | ±100px |
| 评论输入框 | (540, 2200) | ±100px |
| 发送按钮 | (980, 2200) | ±30px |

> **注意**：评论功能需要手机安装 [ADBKeyboard](https://github.com/senzhk/ADBKeyBoard) 才能输入中文，否则 `adb shell input text` 只支持 ASCII 字符。

## ADB 常用命令

```
adb shell input swipe <x1> <y1> <x2> <y2> [duration(ms)]
```

- `x1, y1` — 起始坐标
- `x2, y2` — 终点坐标
- `duration` — 滑动持续时间，单位毫秒
